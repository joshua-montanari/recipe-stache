# Recipe-Stache
A site to share and view recipes!
by: Josh and Derek

Built with: React.js Node.js Express and MongoDB

featuring: react hooks, Context api, axios, material, json web tokens, bcrypt

## Dependencies:
 ### Backend
 express
 mongoose
 dotenv
 cors
 bcrypt
 jsonwebtoken
 
 ### Client
 axios
 @material-ui/core
 @material-ui/icons
 react-router-dom
 react-icons
 react-scripts
